# 项目概述 python541

本项目是一款基于 Python 的学生宿舍管理系统，使用 GUI 界面进行交互，界面采用 tkinter 库进行构建。支持两种数据库：SQL Server 和 MySQL。

# 技术栈

- 编程语言：Python
- 数据库：SQL Server / MySQL
- 界面库：tkinter

# 功能模块

系统主要包括以下功能模块：

1. 学生信息管理
2. 宿舍信息管理
3. 缴费信息管理
4. 报修信息管理

# 系统角色

- 管理员：负责系统的日常管理和维护
- 学生：查询宿舍信息、缴费、报修等

# 运行环境

- 操作系统：Windows/Linux/MacOS
- Python版本：3.x
- 数据库：SQL Server 或 MySQL

# 部署步骤

1. 安装 Python 环境（3.x 版本）
2. 安装 SQL Server 或 MySQL 数据库
3. 导入数据库文件
4. 运行 main.py 文件

# 目录结构

```
.
├── main.py
└── database
    ├── sqlserver.db
    └── mysql.db
```

# 核心亮点

- 简单易懂：只有一个 main.py 文件，易于学习和掌握。
- 使用 Python 的 thinker 框架，简化了数据库操作，便于初学者上手。
